# Image Suite — runtime deps.
#
# This plugin runs inside Wan2GP, so anything Wan2GP already pins is intentionally
# NOT re-listed here. Re-declaring its deps (often unpinned) makes pip re-resolve
# and can break Wan2GP's exact pins — most importantly its custom onnxruntime-gpu
# nightly. Provided by Wan2GP already: gradio, torch, torchvision, Pillow, numpy,
# diffusers, transformers, accelerate, safetensors, huggingface_hub, opencv-python,
# onnx, onnxruntime-gpu, insightface, facexlib.
#
# Auto-installed on first use by core/deps.py (NOT listed here): kornia,
# controlnet_aux, ultralytics.
#
# Optional, deliberately NOT installed here (conflict-prone — pulls protobuf/numpy
# constraints that fight Wan2GP's): mediapipe. It only powers hand preservation
# during body-swap, and the code skips it gracefully when absent. Install manually
# with `pip install mediapipe` if you want that refinement.

# scikit-image: SimilarityTransform for face alignment (core/faceswap.py). The only
# direct dependency Wan2GP doesn't already provide.
